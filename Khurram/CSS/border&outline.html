Borders, Outline & Units Practics

Border

The border property is used to set the border of an element. It can be set using th following properties:

  • border-width
  • border-style
  • border-color
  • border-radius

The border-width property is used to set the width of the border. It can be set using the following values:

  • thin
  • medium
  • thick
  • length

The border-style property is used to set the style of the border. It can be set using the following values:

  • none
  • hidden
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset

The border-color property is used to set the color of the border. It can be set using the following values:

  • color
  • transparent

The border-radius property is used to set the radius of the border. It can be set using the following values:

  • length
  • percentage

Outline

The outline property is used to set the outline of an element. It can be set using the following values:

  • none
  • hidden
  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset

The outline-color property is used to set the color of the outline. It can be set using the following values:

  • color
  • invert
  • transparent

The outline-width property is used to set the width of the outline. It can be set using the following values:

  • thin
  • medium
  • thick
  • length

The outline property is a shorthand property that allows you to set the outline-width, outline-style, and outline-color properties in one declaration.

The outline-offset property is used to set the distance between the outline and the border of an element. It can be set using the following values:

  • length
  • initial
  • inherit

The outline shorthand property is used to set the outline-width, outline-style, and outline-color properties in one declaration.
